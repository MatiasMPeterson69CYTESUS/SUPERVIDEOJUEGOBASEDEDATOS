# TimeSplit (Dragoncito Edition)
Modos Carreras y Fútbol, power-ups (incluye DRAGON), splits (ms), ranking, CSV/Excel, API Sync, música/sonidos WAV.

## Windows (Python 3.13 → usa pygame-ce)
py -m pip install --upgrade pip
py -m pip install -r requirements-windows.txt
py timesplit_game.py

## macOS / Linux (o Windows con Python 3.11)
python -m pip install --upgrade pip
python -m pip install -r requirements.txt
python timesplit_game.py

## Personaje Dragoncito
Si agregas `assets/dragon.png` (48x48, transparente), el juego lo mostrará automáticamente.

## Power-up DRAGON
Aparece con ~10% de probabilidad. En carreras turbo+escudo potentes por 10s; en fútbol disparos ultrafuertes por 10s.
